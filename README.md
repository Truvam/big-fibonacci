# Big Numbers Fibonacci
## Installation:
```
$ git clone https://github.com/Truvam/big-fibonacci.git
$ cd big-fibonacci/ 
$ make
```
## Example:
```
$ ./calc
1: Recursive Fibonacci
2: Optimized Fibonacci
3: Big Numbers
Option: 3
1: Addition
2: Subtraction
3: Multiplication
4: Division
5: Fibonacci

Result: 3361707149818144672666187219454104827980338677164658343636350711365
```
